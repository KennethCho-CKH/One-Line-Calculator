# One-Line-Calculator

### A calculator created with only one line of code

Here are some examples of how to use it:

| Input | Output |
| --- | --- |
| `1 + 1` | `2` |
| `1 - 1` | `0` |
| `1 * 1` | `1` |
| `1 / 1` | `1` |
| `1 == 1` | `True` |
| `1 != 1` | `False` |
| `(1 + 1)-1 != 1 * 1` | `False` |
| `not((1 + 1)-1 != 1 * 1)` | `True` |
